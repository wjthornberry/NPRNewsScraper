What is Web Scraping?

Web Scraping (also termed Screen Scraping, Web Data Extraction, Web Harvesting etc) is a technique employed to extract large amounts of data from websites whereby the data is extracted and saved to a local file on your computer or to a database in table (spreadsheet) format.
